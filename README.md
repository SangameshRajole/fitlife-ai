# 🏋️ FitLife AI — Smart Health Tracker

A full-stack health tracking web application built with React, Node.js, and MongoDB.

## Features
- 🔐 User Authentication (JWT)
- 📊 Dashboard with Health Summary
- ⚖️ BMI Calculator (cm & feet support)
- 💧 Water Intake Tracker
- 🔥 Calories Tracker
- 💪 Workout Tracker
- 🤖 AI Health Suggestions
- 🌙 Dark/Light Mode
- 📱 Mobile Responsive

## Tech Stack
- **Frontend:** React.js, Vite, Tailwind CSS, Framer Motion
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
- **Auth:** JWT, bcryptjs
